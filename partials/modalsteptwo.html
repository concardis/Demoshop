Step 2/4: Create an JSON Order


To perform a request a order hast to be created. You can edit the fields below and the sample JSON will be generated on the right box.


Mandatory Fields

# Parameter Description Value
1 initialAmount Amount as smallest currency unit e.g 100â‚¬ will be result in amount = 10000
2 currency Currency code according to https://en.wikipedia.org/wiki/ISO_4217

{ "initialAmount": {{initialAmount}}, "currency": {{currency}}, "async": { "successUrl": {{successUrl}}, "failureUrl": {{failureUrl}}, "cancelUrl": {{cancelUrl}} }, "transactionType": {{transactionType}} }
