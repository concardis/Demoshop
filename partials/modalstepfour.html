Step 4/4: Initialise the Modal

We are now ready to initialise our modal.
First import the Widget, or download it from here.

<script src="https://pp.payengine.de/widgetjs/payengine.widget.min.js"></script>
Secondly we need to create a function for a callback and the Modal itself. 
  <script>
  //A callback function 
   var callback = function (error, result) { 
     if (error) {
       //There was an error
       //You will find more details in the error object
       return;
     }
     //Handle transaction successfully created
   }

   function openModal() {
     var merchandId = "merchant_kdanvgjnhx";
     var orderId = "order_564c4d97f7";
     PayEngineWidget.initAsModal(merchandId, orderId, callback);
   }
  </script>
We can now add a button to open our modal. 
   <button onclick="openModal()">Open modal</button>
