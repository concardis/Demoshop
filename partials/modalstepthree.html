Step 3/4: Make an order request

Once you got your base64 encoded "MerchandId:ApiKey" and your json order, we are ready to perform our first order request. Here are some examples:


URL https://apitest.payengine.de/v1/orders
HTTP basic authentication header (Step One) Authorization: Basic {{base64}}=
Body (Step Two) {{order}}

CURL can encode base64, so just pass your merchantId and your Api-Key with the -u Flag:

curl -u merchantId:ApiKey --header "Content-Type: application/json" --request POST https://apitest.payengine.de/v1/orders --data '{{order}}'
var xhr = new XMLHttpRequest(); var url = "https://apitest.payengine.de/v1/orders"; //Payengine API Orders Endpoint xhr.open("POST", url, true); //Post request xhr.setRequestHeader("Content-Type", "application/json"); //Content-Type: Json xhr.setRequestHeader("Authorization","Basic {{base64}}"); //Authorization in base64 var data = JSON.stringify({{order}}); //Our Order xhr.send(data);
